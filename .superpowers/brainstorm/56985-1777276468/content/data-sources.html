Data source landscape

Pick everything you're willing to use. Some are already wired into the project.

✓

Capitol Trades API FREE

Congressional STOCK Act disclosures. Every trade filed by House + Senate members. Updated within hours of filing. Also covers some executive branch officials.

⏱ 45-day max delay (STOCK Act window) · Coverage: all ~535 members · No key required

✓

GDELT FREE

Global Database of Events, Language and Tone. Monitors news across 100+ languages, updated every 15 minutes. Covers politics, business leaders, geopolitics, influential figures worldwide.

⏱ 15-min lag · Massive coverage · BigQuery or REST API

★

Polygon News ALREADY INTEGRATED

Already wired in for ticker-scoped news. Can be extended to cover political and macro stories without any new API key.

Extend existing integration · No extra cost

~

StockTwits API FREE TIER

Real-time social sentiment per ticker. Bullish/bearish ratios, message volume, trending tickers. Good proxy for retail sentiment without X/Twitter costs.

Free: 200 req/hr · Paid: $99/month for higher limits

~

Reddit API FREE

r/wallstreetbets, r/stocks, r/investing mention counts and sentiment. Free tier is generous enough for this use case. Good for meme/retail momentum signals.

Free: 100 req/min · No paid tier needed for this use case

~

NewsAPI $449/MONTH

Broad news coverage across business, politics, technology. Free dev tier: 100 req/day, no live data. Real-time requires paid plan.

Dev: free (delayed) · Business: $449/mo (real-time)

~

QuiverQuant $50/MONTH

Aggregates congressional trades, lobbying data, government contracts, and political Twitter sentiment into clean structured feeds. Easier than raw STOCK Act scraping.

$50/mo · Congress + lobbying + govt contracts in one API

$$$

X / Twitter API ~$5,000/MONTH

Real-time tweets from politicians and executives. Highest signal density but prohibitively expensive for retail. Basic tier ($100/mo) is too rate-limited for live trading.

Basic: $100/mo (very limited) · Enterprise: ~$5k/mo

Recommended baseline — $0/month

Capitol Trades + GDELT + Polygon News (already wired) + StockTwits + Reddit covers all four intelligence layers for free. Add QuiverQuant ($50/mo) for cleaner congressional data with lobbying and government contract context.
